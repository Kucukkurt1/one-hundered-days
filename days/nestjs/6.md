## üìö Dictionary for Developers

*   **ORM (Object-Relational Mapper):** A library that provides a way to interact with a database using an object-oriented paradigm.
*   **TypeORM:** A popular ORM for TypeScript and JavaScript that can be used with NestJS.
*   **Entity:** A class that maps to a database table.
*   **Repository:** A class that provides methods for interacting with a specific database table (e.g., `find`, `save`, `delete`).
